## 1. Conv2D 要点

```python
# 单通道核心
for i in range(out_h):
    for j in range(out_w):
        region = image[i*stride:i*stride+kh, j*stride:j*stride+kw]
        output[i, j] = np.sum(region * kernel)
```

**必说：**
- 输出尺寸公式  
- 多通道：对 \(C_{in}\) 累加再加 bias  
- DL 不翻转核  

详见 `conv2d.py` 与 [03-算法数学公式.md](./03-算法数学公式.md#1-conv2d二维卷积)

## 2. NMS 要点

```python
order = scores.argsort()[::-1]
keep = []
while len(order) > 0:
    i = order[0]
    keep.append(i)
    ious = [compute_iou(boxes[i], boxes[j]) for j in order[1:]]
    order = order[1:][np.array(ious) <= threshold]
```

**必说：**
- IoU 公式  
- 按类 NMS  
- 训练不用、推理用

## 3. 双线性插值要点

```python
x0, y0 = int(np.floor(x)), int(np.floor(y))
dx, dy = x - x0, y - y0
return (1-dx)*(1-dy)*v00 + dx*(1-dy)*v01 + (1-dx)*dy*v10 + dx*dy*v11
```

**必说：**
- 四权重之和为 1  
- resize 的 center-aligned 映射  
- grid_sample 的 [-1,1] 转换  

详见 `bilinear_interp.py` 与 [03-算法数学公式.md](./03-算法数学公式.md#3-双线性插值bilinear-interpolation)
