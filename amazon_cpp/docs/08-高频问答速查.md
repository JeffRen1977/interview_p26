# Amazon C++ 高频问答速查（一页纸）

> 详细解释见各 Part 文档；示例见 [`examples/`](../examples/)

## 内存与对象

| 问题 | 一句话答案 |
|------|-----------|
| `new` vs `malloc` | new 类型安全+调构造；malloc 只分配字节 |
| `delete` vs `free` | delete 析构+释放；free 只释放 |
| Stack vs Heap | 栈自动、快、小；堆手动、慢、大 |
| RAII | 构造获取资源，析构释放；异常安全 |
| Rule of Five | 有析构管理资源时，需拷贝/移动构造+赋值共 5 个 |
| Move Constructor | 窃取资源；临时量、std::move、（无 RVO 时）返回值 |
| `std::move` | 只是 cast 到 rvalue，不移动；触发移动重载 |
| Placement `new` | 在已有内存上构造，不分配 |

## 指针与 const

| 问题 | 一句话答案 |
|------|-----------|
| `const int*` | 所指不可改 |
| `int* const` | 指针不可改 |
| `nullptr` vs `NULL` | nullptr 类型安全，无 int 歧义 |
| `void*` | 不能解引用，不知 size/alignment |

## 智能指针

| 问题 | 一句话答案 |
|------|-----------|
| `unique_ptr` vs `shared_ptr` | 独占 vs 共享引用计数 |
| `shared_ptr` 实现 | raw ptr + control block（strong/weak count） |
| `weak_ptr` | 不增加 strong count；打破循环引用 |
| Circular Reference | A↔B 互持 shared_ptr 泄漏 → 一方改 weak_ptr |
| `enable_shared_from_this` | 在已被 shared_ptr 管理时安全返回 shared_ptr<this> |

## 多态

| 问题 | 一句话答案 |
|------|-----------|
| vtable | 每类一张虚函数指针表 |
| vptr | 对象开头隐藏指针，指向 vtable |
| virtual 开销 | 间接跳转 + 难内联 + 每对象 8B |
| Virtual Destructor | 基类指针 delete 派生对象必须 virtual |

## 内存布局

| 问题 | 一句话答案 |
|------|-----------|
| `sizeof({char; int})` | 通常 8：对齐 padding |
| Alignment | 成员 offset 须为 alignof 倍数 |

## STL

| 问题 | 一句话答案 |
|------|-----------|
| vector 扩容 | 通常 2×；均摊 O(1) push_back |
| `reserve` vs `resize` | reserve 只扩 capacity；resize 改 size |
| `push_back` vs `emplace_back` | emplace 原地构造，少一次移动 |
| unordered_map | 哈希桶+链；rehash 当 load factor 过高 |
| map O(log n) | 红黑树，树高 log n |
| deque push_front O(1) | 分段 chunk，两端增长 |
| SSO | 短 string 内联 buffer，不堆分配 |

## 并发

| 问题 | 一句话答案 |
|------|-----------|
| mutex vs atomic | mutex 阻塞等待；atomic 硬件原子指令 |
| mutex vs spinlock | 长临界区用 mutex；极短忙等 spinlock |
| condition_variable | 必须配 mutex；用 while 防虚假唤醒 |
| CAS | 比较并交换；无锁基础 |
| ABA | 指针值相同但中间被改过；用版本号 |
| memory_order acquire/release | release 写同步到 acquire 读 |
| False Sharing | 不同核写同一 cache line → padding 分离 |

## 类型转换

| cast | 用途 |
|------|------|
| `static_cast` | 相关类型、上行转型 |
| `dynamic_cast` | 多态下行，运行时检查 |
| `const_cast` | 增删 const |
| `reinterpret_cast` | 位级重解释，危险 |

|------|-----------|
